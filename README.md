# vue-egg-sql_Message-Board
vue+egg+sql简易留言板 增加删除更改功能  用axios前端发送数据在数据库保存 后端读取数据发送给前端

学习用 其他概不负责

版本
"axios": "^0.20.0",
"core-js": "^3.6.5",
"vue": "^2.6.11"
"egg": "^2.15.1",
"egg-cors": "^2.2.3",
"egg-mysql": "^3.0.0",
"egg-scripts": "^2.11.0"

运行
inti
  npm i -> npm run serve

server
  npm i -> npm run dev
